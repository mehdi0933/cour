Parcours d'éléments

Le DOM est un modèle objet.

Il permet d'interagir avecn'importe quelle partie du document.

Il existe de nombreux moyens de parcourir le document. Avec les noms de balises, avec les attributs, les ID.

Néanmoins, attention, le DOM n'est pas implémenté de la même manière dans tous les navigateurs. Certains navigateurs comme IE6 et IE7 ne respectent pas la norme.
